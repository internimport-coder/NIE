Setup Google Cloud (Sekali Saja)
  1. Buka console.cloud.google.com → buat project baru
  2. Enable Google Drive API
  3. Buat OAuth 2.0 Client ID (tipe: Web App) → salin Client ID
  4. Buat API Key → salin API Key
  5. Isi CLIENT_ID dan API_KEY di bagian GDRIVE_CONFIG di app.js
Cari Manual (File Selain NIE)
Untuk cari file selain NIE — nama produk, nomor invoice, nama dokumen, dll. Paste banyak kata kunci sekaligus, satu per baris. Box NIE di atas khusus untuk nomor NIE saja.
