Google Cloud Setup (One-Time)
  1. Open console.cloud.google.com → create a new project
  2. Enable Google Drive API
  3. Create an OAuth 2.0 Client ID (type: Web App) → copy the Client ID
  4. Create an API Key → copy the API Key
  5. Fill in CLIENT_ID and API_KEY in the GDRIVE_CONFIG section of app.js
